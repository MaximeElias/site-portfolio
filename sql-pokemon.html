Explication du Projet

Ce projet consiste en la création d'une base de données SQL pour les 151 premiers Pokémon issus des jeux Pokémon Rouge et Bleu. L'objectif est de modéliser les relations entre les Pokémon, leurs types, évolutions, et statistiques, en utilisant SQL pour effectuer des requêtes complexes et manipuler les données efficacement.

Voir le code source

Améliorations à Venir

Pour rendre la base de données plus complète, je prévois d'ajouter des fonctionnalités supplémentaires comme la gestion des capacités et des objets, ainsi que des requêtes pour filtrer les Pokémon par type, puissance, ou génération. Je compte aussi optimiser les requêtes pour des performances accrues lors des recherches dans de grands ensembles de données.

Conclusion

Ce projet m'a permis de perfectionner mes compétences en SQL et en gestion de bases de données. Grâce à ce mini-projet, j'ai pu comprendre les concepts de modélisation relationnelle, de normalisation des tables, et de manipulation de données. Il s'agit d'une première étape dans la création d'un Pokédex interactif avec des données enrichies sur chaque Pokémon.
